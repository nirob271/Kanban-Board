# 📌 Kanban Board

A fully functional **Kanban Board** built using **HTML, CSS, and Vanilla JavaScript**. This project was developed as a JavaScript practice project to strengthen my understanding of DOM manipulation, event handling, dynamic UI updates, and the HTML5 Drag & Drop API.

Users can create tasks, move them across different workflow stages using drag and drop, and manage their progress visually through a clean and intuitive interface.

---

## 🚀 Features

- ✅ Create new tasks
- ✅ Delete tasks
- ✅ Drag and Drop tasks between columns
- ✅ Organize tasks into workflow stages:
  - To Do
  - In Progress
  - Done
- ✅ Real-time task count updates
- ✅ Dynamic DOM manipulation
- ✅ Responsive and clean user interface
- ✅ Built entirely with Vanilla JavaScript (No frameworks)
